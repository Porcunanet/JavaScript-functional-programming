JavaScript Functional Programming

A quick reference for common array methods and functional patterns.

Check Even/Odd

const checkEven = n => n % 2 === 0 ? "even" : "odd";

Count Odds

const numOfOdds = (num) => Math.floor((num + 1) / 2);

1. MAP

Transforms each element in an array using a function. Returns a new array of the same length.

// Double numbers
const doubled = numbers.map(n => n * 2);

// Extract properties
const names = users.map(user => user.name);

2. FILTER

Returns a new array containing only the elements that pass the specified test.

// Get even numbers
const evens = numbers.filter(n => n % 2 === 0);

// Filter by property
const engineers = users.filter(u => u.dept === 'Engineering');

3. REDUCE

Reduces an array to a single value (number, object, or new array) by applying an accumulator function.

// Sum all numbers
const sum = numbers.reduce((acc, curr) => acc + curr, 0);

// Group by category
const depts = users.reduce((acc, user) => {
  (acc[user.dept] = acc[user.dept] || []).push(user.name);
  return acc;
}, {});

Chaining & Composition

Combine multiple methods to perform complex data transformations in a readable way.

The Pipeline Pattern

const pipeline = (...fns) => val => fns.reduce((v, f) => f(v), val);

const result = pipeline(addTen, multiplyByTwo, subtract5)(10); 
// Result: 35
Pro Tip: Methods like find, some, and every short-circuit, meaning they stop iterating as soon as the condition is met!
